# Campus Traffic Model

Python-модель інтелектуального розподілу трафіку між Wi-Fi точками доступу та Ethernet-сегментами кампусної мережі.

## Що моделюється

- Кампусна мережа з Wi-Fi точками доступу та Ethernet-сегментами.
- Різні типи трафіку: відео, голос, LMS, веб, файли, фонові оновлення.
- 4 алгоритми розподілу трафіку:
  1. Baseline — базове підключення за зоною.
  2. Least Loaded — вибір найменш завантаженого сегмента.
  3. QoS Priority — вибір з урахуванням класу трафіку.
  4. Intelligent — багатокритеріальний інтелектуальний алгоритм.
- 5 сценаріїв експерименту.
- Порівняння за затримкою, втратами, завантаженням, QoS-задоволенням та інтегральною ефективністю.

## Запуск

```bash
pip install -r requirements.txt
python main.py
```

Після запуску результати зберігаються у папці `results/`:

- `summary_results.csv` — зведена таблиця результатів;
- `flows_distribution.csv` — розподіл потоків;
- `delay_comparison.png` — порівняння середньої затримки;
- `qos_satisfaction.png` — частка потоків, обслужених відповідно до QoS;
- `overloaded_segments.png` — кількість перевантажених сегментів;
- `efficiency_index.png` — інтегральна ефективність алгоритмів;
- `topology.png` — схема кампусної мережі.
